Letter Game

After choosing a language and a number within the ranges below, the program will print out a letter!

 

Language Range
"English" 1-26
"Spanish" 1-27
"Igbo" 1-33
"Yoruba" 1-25
"Ijaw" 1-26

 

Instructions: Select a language from the dropdown box below. In the box, enter a number within the given range corresponding to the language you chose.
